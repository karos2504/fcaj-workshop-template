### Week 2 Objectives

* Master Amazon Simple Storage Service (S3) object storage, Storage Classes, and automated S3 Lifecycle data rules.
* Understand S3 Bucket Policies, SSE Encryption, and static website hosting implementation.
* Configure Amazon CloudFront CDN to accelerate web content distribution, explore Edge Computing (Lambda@Edge), and manage resources using AWS CLI & AWS Cloud9.

### Week 7 Objectives

* Understand Infrastructure as Code (IaC) principles using AWS CloudFormation & AWS CDK v2 (TypeScript/Python).
* Learn Containerization technology using Docker (Dockerfile, Multi-stage builds) and private container image management on Amazon ECR.
* Deploy containerized microservices using Amazon ECS on Serverless AWS Fargate and explore Amazon EKS (Kubernetes) fundamentals.

* Proficient in object-oriented infrastructure modeling using AWS CDK v2.
